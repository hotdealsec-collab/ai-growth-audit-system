# AI-assisted Growth Audit System

This project demonstrates a Growth Architecture framework that evaluates:

- Acquisition efficiency
- Activation signals
- Retention stability
- Monetization performance
- Capital efficiency
- Measurement reliability

The system produces:

- Growth Health Score
- Measurement Confidence Score
- Bottleneck Detection
- Capital Allocation Suggestions
- Final Growth Recommendation

## Demo

Upload a CSV containing campaign performance metrics and receive an automated growth audit.
